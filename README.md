# wow-hide-character-frame

Simple addon that hides your character frame when:
- Out of combat
- No target
- Full health
- Power conditions:
 - Mana is full
 - Energy is full
 - Focus is full
 - Rage is zero
- No debuffs

Frame shows when:
- In combat
- Target selected
- Health not full
- Power not full (or rage above zero)
- Any debuff present

No setup required - just install and play.
